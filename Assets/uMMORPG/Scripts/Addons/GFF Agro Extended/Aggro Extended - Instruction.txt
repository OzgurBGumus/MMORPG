                                      Installation and Configuration

1. Edit the "MonsterBrain.cs"  Script in “/Scripts/ScriptableBrains/" 
  a) before line     public class MonsterBrain
       change to     public partial class MonsterBrain

  
  b) in void UpdateClient    after line     Monster monster = (Monster)entity;
                               add line     Utils.InvokeMany(typeof(MonsterBrain), this, "UpdateClient_", monster);
		 

		 in 3 locations
  c)  after line     if (EventDeathTimeElapsed(monster)) {} // don't care
        add line     if (EventAggroExtendedEnd(monster)) FindNewAggroTarget(monster);


2. Edit the "Monster.cs"  Script in “/Scripts/" 
   a) find line      target = entity;
     replace to      UpdateAggroList(entity, aggroValueMax);

   b) find and comment line     if (newDistance < oldDistance * 0.8) target = entity;


3. Move folder "GFF Aggro" from  (GFF Aggro Extended/Skills/) to Resources/Skills, Buffs, Status Effects/


4. Edit Warrior prefab
  a) Add new skills "Aggro Area", "Aggro Target" and "Agro Drop" in script "Skills" to skill templates
  b) in Component Animator Set Controller to "Controller with Aggro Skills"


5. a) Add to scene new "Skeleton" prefab
      (new prefabs added a sprite - an indicator of aggression)
