                                     Installation and Configuration

//if you do not use our Inventory addon
1. Edit the "UIInventory.cs"  Script in   Assets/uMMORPG/Scripts/_UI/
  a) after line     slot.amountText.text = itemSlot.amount.ToString();
      add lines     slot.GetComponent<Image>().sprite = player.itemRarityConfig.GetSprite;
                    slot.GetComponent<Image>().color = player.itemRarityConfig.GetColor(itemSlot.item);

  b) after line     slot.amountOverlay.SetActive(false);
      add lines     slot.GetComponent<Image>().sprite = player.itemRarityConfig.GetSprite;
                    slot.GetComponent<Image>().color = player.itemRarityConfig.GetColorForEmptySlot();


//if you do not use our Equipment addon
2. Edit the "UIEquipment.cs"  Script in   Assets/uMMORPG/Scripts/_UI/
  a) after line     slot.amountText.text = itemSlot.amount.ToString();
      add lines     slot.GetComponent<UnityEngine.UI.Image>().sprite = player.itemRarityConfig.GetSprite;
                    slot.GetComponent<UnityEngine.UI.Image>().color = player.itemRarityConfig.GetColor(itemSlot.item);

  b) after line     slot.amountOverlay.SetActive(false);
      add lines     slot.GetComponent<UnityEngine.UI.Image>().sprite = player.itemRarityConfig.GetSprite;
                    slot.GetComponent<UnityEngine.UI.Image>().color = player.itemRarityConfig.GetColorForEmptySlot();


3. Edit all player prefabs
  a) in Inspector In script "Player" in field "Item Rarity Config" select "ItemRarityConfig" from (Assets/uMMORPG/Scripts/Addons/GFF Item Rarity/Resources/)



 How to adjust the rarity of items 
  * all files in a folder (uMMORPG\Resources\Items\) parameter added "Rarity"
