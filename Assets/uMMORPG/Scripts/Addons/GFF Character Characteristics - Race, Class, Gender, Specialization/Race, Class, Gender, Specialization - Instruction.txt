                                      Installation and Configuration

1. Edit the "EquipmentItem.cs"  Script in “Assets/uMMORPG/Scripts/ScriptableItems/" 
   a)   Find line     public class EquipmentItem : UsableItem
        change to     public partial class EquipmentItem : UsableItem

   b) Before line     return tip.ToString();
         add line     Utils.InvokeMany(typeof(EquipmentItem), this, "ToolTip_", tip);


2. Edit the "UsableItem.cs"  Script in “Assets/uMMORPG/Scripts/ScriptableItems/" 
   a)   Find line     public abstract class UsableItem : ScriptableItem
        change to     public abstract partial class UsableItem : ScriptableItem



3. Edit the "Database.cs" Script in “ Assets\uMMORPG\Scripts " 
   a)  after line     public bool deleted { get; set; }
        add lines     public RaceList race { get; set; }
                      public string gender { get; set; }
                      public string specialisation1 { get; set; }
                      public string specialisation2 { get; set; }
